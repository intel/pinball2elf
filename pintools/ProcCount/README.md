# Tool for RTN profiling across forks/execs
## Tool knobs
### -msgfile  [default msg.out]
        tool messages
### -outdir  [default .]
        Output directory

## Relevant Pin kinb
 Also use the pin  knob '-follow_execv"
### -follow_execv  [default 0]
  Execute with Pin all processes spawned by execv class system calls
